A tool to test you on any taxons that are found on iNaturalist. Default list is the 2025 Science Olympiad Entomology list. Idea from SciolyID.

Note: this tool uses the browser's local storage to more quickly access images.

To create a list, enter in all of the names of the taxon you wish to be tested on. If you need to clarify taxa, paste the number iNaturalist taxon id after the taxa name with a space and % sign preceding it.
